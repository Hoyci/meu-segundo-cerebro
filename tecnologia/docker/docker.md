Docker é uma plataforma open-source para desenvolvimento e execução de sistemas de softwares.
Docker permite que você separe sua aplicação da sua infraestrutura para que você possa entregar software de forma rápida. Usando as metodologias para entregar, testar e implementar código, você pode reduzir de forma significativa o delay entre escrever código e rodar em produção

Docker permite que o desenvolvedor quebre uma aplicação em pequenas partes chamadas de containers onde cada parte tem uma única responsabilidade.

Docker é uma ferramente que coloca a sua aplicação em uma caixinha, chamada de [[container]], que pode ser executada em qualquer ambiente, independente das configurações do sistema
