Docker é uma ferramente que coloca a sua aplicação em uma caixinha, chamada de [[container]], que pode ser executada em qualquer ambiente, independente das configurações do sistema.

Deve-se usar docker porque:
* Consistência: Docker garante que o software irá rodar da mesma maneira em qualquer ambiente.
* Eficiência: Container usa menos recurso do que máquinas virtuais
* Velocidade: É mais rápido para desenvolver, testar e implementar software
* Facilidade: Simplifica o processo de configuração do ambiente de desenvolvimento
