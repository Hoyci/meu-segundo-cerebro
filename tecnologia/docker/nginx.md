NGINX é um software para servidores web que também pode ser usado como proxy reverso, balanceador de carga, cache HTTP e outras coisas. É conhecido por sua alta performance, estabilidade e eficiência no uso de recursos.

Suas principais funções são:
* Servir páginas web estáticas.
* Proxy reverso: Recebe solicitações dos clientes e as encaminha para outros servidores.
* Balanceador de carga: distribui o tráfego entre vários servidores para evitar sobrecarga e melhorar a escalabilidade
* Cache HTTP: Armazena respostas de solicitações para acelerar o tempo de resposta e reduzir a carga nos servidores.

NGINX usa uma arquitetura assíncrona baseada em eventos, permitindo gerenciar muitas conexões simultâneas com poucos recursos.
