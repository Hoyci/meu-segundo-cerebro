Docker CLI é uma aplicação de linha de comando usada para interagir com o docker daemon. Ou seja, é uma aplicação que me permite buildar,  baixar e rodar imagens docker.

Comandos básicos:
* docker pull: Esse comando baixa uma imagem do docker registery
* docker build: Esse comando builda uma imagem de acordo com um dockerfile
* docker run: Esse comando roda um container a partir de uma imagem
* docker ps: Esse comando lista todos os containers
* docker images: Esse comando lista todas as imagens
* docker stop: Esse comando para um container
* docker rm: Esse comando remove um container
* docker rmi: Esse comando remove uma imagem
* docker push: Esse comando faz push de uma imagem para uma cloud ou para o [[docker hub]]
